# 📊 Sales Data Analysis

## 📌 Objective
Analyze sales data to identify trends, high-performing products, and business opportunities.

## 📂 Dataset
Contains information about:
- Product
- Sales
- Profit
- Quantity
- Region

## 🔍 Key Analysis
- Region-wise performance
- Product performance
- Sales vs Profit relationship
- Demand analysis using Quantity

## 📊 Key Insights
- Camera and Monitor are top-performing products.
- Mouse, Printer, and Smartphone show high profit with growth potential.
- North region shows weaker performance.
- Not all sales are profitable.

## 🚀 Conclusion
The company should focus on promoting high-profit products with moderate sales while improving performance in underperforming regions.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
